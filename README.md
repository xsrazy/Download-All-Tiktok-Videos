# Download All Tiktok Videos
Simple script to Download All Tiktok Videos without watermark

That script only does one thing if you want many downloads options consider looking at [Xsrazy](https://xsrazy.github.io/)
- Script Download All Tiktok Videos of a single user profile and the descriptions.

Note: Videos are stored in @username folder whereas the description is stored in names.txt

Warning: Since Download All Tiktok Videos, it can be alot of data depending on the profile you enter or the number of videos a user has. 

# requirements:
+ Git
+ Node Js

# Installtion:
After installing Git and Node Js

+ git clone https://github.com/xsrazy/Download-All-Tiktok-Videos

+ cd Download-All-Tiktok-Videos

navigate to the project directory and use this command:

+ npm i

+ node allvideouser.js address: tiktok profile

Example: node allvideouser.js https://www.tiktok.com/@jkt48.official?is_from_webapp=1&sender_device=pc (Replace with your username, just username)

# Warning!!
Of course you can change the username jkt48.official to whatever user you want 
to download all videos of single user.

Results will be saved in names.txt and videos in username folder location.

All videos will be saved without watermark unless the user reuploaded his videos with logo.
